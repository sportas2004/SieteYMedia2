# Siete y Media

Proyecto Kotlin Desktop con Jetpack Compose.

## Estructura

- `composeApp/src/jvmMain/kotlin/org/example/project/main.kt`: punto de entrada de la aplicacion.
- `composeApp/build.gradle.kts`: configuracion del modulo desktop.

## Ejecutar

En Windows:

```powershell
.\gradlew.bat :composeApp:run
```

## Compilar

```powershell
.\gradlew.bat :composeApp:build
```
